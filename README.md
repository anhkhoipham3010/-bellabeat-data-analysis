# Bellabeat Data Analysis Case Study

## Project Overview

This project analyzes smart device fitness data to identify trends in user behavior and provide recommendations for Bellabeatâ€™s marketing strategy.

Bellabeat is a wellness technology company that produces smart health devices for women. The company wants to understand how consumers use smart devices to improve marketing and product strategy.

---

## Business Task

The goal of this project is to analyze smart device fitness data and identify usage trends that can help Bellabeat improve its marketing strategy.

Key Questions:

1. What are some trends in smart device usage?

2. How could these trends apply to Bellabeat customers?

3. How could these trends help influence Bellabeat marketing strategy?

---

## Dataset

Dataset used:

Fitbit Fitness Tracker Data

Source:
https://www.kaggle.com/datasets/arashnic/fitbit

The dataset contains:

- Daily activity
- Steps
- Calories burned
- Sleep tracking
- Heart rate data

Total users: 30 Fitbit users
